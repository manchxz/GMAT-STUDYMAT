Why algebraic logic, not algebraic computation

The GMAT is not a high-school algebra test. It does not reward you for solving cleanly when you could have side-stepped the algebra entirely. Almost every "algebra question" on the Focus Edition is really one of three things in disguise: an inequality whose solution is a region, a quadratic whose solution is a pair, or an expression that simplifies in one specific algebraic move. Once you classify, the answer follows in seconds.

This chapter teaches you the classification reflex and the three or four moves that resolve almost every question. You will spend more time learning when not to do algebra than learning algebra itself.

The five rules that solve 90% of algebra questions

  • Equation do the same thing to both sides. Symmetry is preserved.
  • Inequality do the same thing to both sides — but if you multiply or divide by a negative, flip the sign.
  • |x| = a → x = a or x = −a. Absolute value is distance from zero.
  • ax² + bx + c = 0 → factor if possible; otherwise discriminant b² − 4ac tells you how many real solutions exist.
  • Sign chart for any inequality with multiple factors: place all roots on a number line, test each region, take the answer set.

Variables and their hidden domains

A variable is a placeholder for a number. But on the GMAT, the placeholder always comes with a hidden constraint — "positive integer", "real number", "non-zero", "between 0 and 1" — and the constraint is often where the question lives.

ELI5 · two simple examples

Example 1 · the lunch order. A waiter says, "I have a number in mind, between 1 and 5, and it's a whole number." That is exactly what a constrained variable is. There are only five possibilities, not infinity. Knowing the domain (whole, between 1 and 5) shrinks the world the variable could live in.

Example 2 · the locker padlock. A padlock has a 3-digit combination, each digit from 0 to 9. Each digit is a variable, but its domain is the set {0, 1, 2, …, 9}. You would never test "the digit is 14" — the constraint forbids it. GMAT variables come with constraints exactly like this. Always read the line that defines them before you start solving.

Expert · the constraint checklist

For every variable in a problem, log:

  1. Type — integer, real, positive integer, non-negative real, etc.
  2. Range — bounded above? bounded below? open or closed?
  3. Forbidden values — denominator non-zero, log argument positive, even-root argument non-negative.

The most-missed constraints on the Focus Edition: "positive integer" excludes zero (zero is non-negative, not positive); "non-zero" allows fractions; "real" allows irrationals.

Equations and the symmetry of operations

An equation is a balance scale. Whatever you do to one side, you do to the other. Add 5 to both sides: still balanced. Multiply both sides by 7: still balanced. Take both sides to the power of 3: still balanced. The single golden rule of equation-solving is symmetry preservation.

ELI5 · two simple examples

Example 1 · the seesaw. Imagine a seesaw, perfectly balanced, with three apples on the left and three apples on the right. If you take one apple off the left, you must take one off the right or the seesaw tips. That is solving for a variable: every operation is one move on each side, keeping the balance.

Example 2 · the twin bank accounts. Two siblings have identical bank accounts: $300 each. Their parents say "we will always credit or debit you the same amount." If $50 is added to one, $50 is added to the other. Whatever happens to one side of the equation, the same operation happens to the other. The accounts (sides) stay equal.

Trap · squaring both sides

Squaring both sides of an equation can introduce solutions that did not satisfy the original. Example: x = −3 squared gives x² = 9, which has solutions x = ±3. The +3 is an extraneous solution. After squaring, always plug your answers back into the original to verify. See the Extraneous Solution Trap.

Inequalities — and the one rule that breaks them

An inequality behaves like an equation 95% of the time. The 5% that breaks it is the rule everyone forgets in the middle of a problem.

The sign-flip rule

  • Multiply or divide both sides by a positive → sign stays the same.
  • Multiply or divide both sides by a negative → sign flips: < becomes >, and vice versa.
  • If the variable's sign is unknown → you may not multiply or divide by it. Period.
ELI5 · two simple examples

Example 1 · the elevator analogy. Going up an elevator, the higher floor stays higher. Now imagine the elevator flips upside down. The floor that was on top is now on the bottom. Multiplying by a negative is exactly that flip — what was bigger is now smaller. The order reverses.

Example 2 · scoring in a race. If runners A and B finish the race and A is faster (a smaller time), then A < B. Now imagine you score the race by "minus their time" instead. A's score is bigger because his negative time is closer to zero. The same comparison, multiplied by −1, flips. That is exactly the inequality sign-flip.

Trap · multiplying by a variable of unknown sign

If you have 1/x < 5 and multiply both sides by x, you cannot conclude 1 < 5x — because x might be negative, in which case the sign should have flipped. Whenever a variable's sign is not known, do not multiply or divide by it. Use a sign chart instead. See Sign analysis, below.

Absolute value — distance, not magic

Students fear absolute-value problems more than they should. The fear comes from treating | · | as a special operator with mysterious rules. It is not. Absolute value is just distance from zero.

ELI5 · two simple examples

Example 1 · steps from the front door. If your house's front door is at position 0 and you walk 3 steps in either direction, you have walked a distance of 3. |3| = 3 and |−3| = 3 both mean "3 steps from the front door". Distance does not care about direction.

Example 2 · GPS distance. Your phone says you are 5 km from home. That number does not tell you whether you are north or south — only that you are 5 km away. |x| = 5 says exactly the same thing: x is 5 units from zero, but the sign is open. So x = 5 or x = −5.

Expert · the three forms you will actually see
  1. |x| = a (with a ≥ 0) → x = a or x = −a.
  2. |x| < a → −a < x < a. Inside the band.
  3. |x| > a → x < −a or x > a. Outside the band.

For |x − k|, just replace "distance from 0" with "distance from k". |x − 4| < 3 means x is within 3 of 4, i.e. 1 < x < 7.

Quadratics — three forms, one game

A quadratic is any expression of the form ax² + bx + c, with a ≠ 0. On the GMAT, you will face quadratics in three structural forms. Recognize the form, take one move, and you are done.

The three forms of a quadratic

  • Factored: (x − r)(x − s) = 0 → roots are r, s.
  • Standard: ax² + bx + c = 0 → sum of roots = −b/a; product of roots = c/a.
  • Vertex / completed square: (x − h)² + k → minimum (or maximum) at x = h.

Discriminant: D = b² − 4ac.
D > 0 → two real roots. D = 0 → one repeated real root. D < 0 → no real roots.

ELI5 · two simple examples

Example 1 · the U-shaped slide. A quadratic looks like a U-shaped slide on a graph. The two roots are the spots where the slide touches the ground. If the slide is high above the ground, it never touches — that is "no real roots". If the slide just kisses the ground in one spot, that is "one repeated root". If it dips below the ground, it crosses in two places — two roots.

Example 2 · two prices that multiply to 12. If (x − 4)(x − 3) = 0, the only way for the product of two prices to be zero is if at least one price is zero. So x − 4 = 0 (giving x = 4) or x − 3 = 0 (giving x = 3). That is the entire factoring trick: a product is zero exactly when one of the factors is zero.

Expert · the patterns to recognize on sight 
(x + a)² = x² + 2ax + a²
(x − a)² = x² − 2ax + a²
(x + a)(x − a) = x² − a²        ← the difference of squares
(x + a)(x + b) = x² + (a+b)x + ab

The fourth pattern is the key to most GMAT factoring questions. To factor x² + 7x + 12, ask: which pair of integers multiplies to 12 and adds to 7? Answer: 3 and 4. So x² + 7x + 12 = (x + 3)(x + 4).

Sign analysis — the sign-chart method

When an inequality involves a product or a quotient of factors, do not solve algebraically. Use a sign chart. It is faster, more reliable, and protects you from the multiply-by-unknown-sign trap.

Sign-Chart Procedure
Inequality with product or quotient of factors
↓
1 · Find roots of every factor (where each = 0)
↓
2 · Mark roots on a number line. They split it into regions.
↓
3 · Pick a test value in each region. Check the sign of the full expression.
↓ Result
Take the union of regions whose sign matches the inequality.
Expert · worked sign chart

Solve (x − 1)(x + 2) > 0.

Roots:  x = 1,  x = −2.
Regions: (−∞, −2),  (−2, 1),  (1, ∞).
Test:    x = −3:  (−4)(−1) = +4   → positive ✓
         x = 0:   (−1)(+2) = −2   → negative ✗
         x = 2:   (+1)(+4) = +4   → positive ✓
Solution: x < −2  or  x > 1.

This generalizes: any inequality with two or more factors collapses to a sign chart. Three factors make three roots and four regions. The structure is identical.

Smart substitution — when to put the algebra down

For "must be true" or "could be true" problems, plugging in numbers is often faster and safer than algebra. The art is in knowing when to substitute, what numbers to choose, and how many cases to test.

ELI5 · two simple examples

Example 1 · the toy machine. Imagine a toy machine where you put in a coin and out comes a different coin, but you do not know the rule. Instead of guessing the rule, you just try a few coins and see what comes out. Same with algebra: instead of solving "what does 3a + 2 evaluate to?", just try a = 1, then a = 2, and see the pattern.

Example 2 · the test-tasting at a bakery. Before buying a whole cake, you taste a small sample. The sample is your "test value". On the GMAT, plugging in x = 2 or x = −1 is like sampling — it tells you whether a statement is true without baking the whole cake.

Expert · the substitution discipline

When substituting:

  1. Pick small, distinct, easy values — usually 0, 1, 2, −1 — that obey every constraint of the problem.
  2. Test multiple categories. If the variable could be positive or negative or zero, test all three. If it could be integer or fraction, test both.
  3. Use values that make the answer choices distinguishable. If picking 1 makes three answer choices evaluate to the same number, pick something else.
  4. Stop substituting once one choice fails. "Must be true" eliminations are absolute.

Worked examples

Worked Example · 01 · Inequality with sign-flip

Solve −3x + 7 > 1 for x.

  1. Subtract 7: −3x > −6.
  2. Divide both sides by −3 — the divisor is negative, so flip the sign.
  3. Result: x < 2.

Answer: x < 2.

Worked Example · 02 · Quadratic by recognition

If x² − 5x − 14 = 0, what are the values of x?

  1. Standard form. Look for two integers whose product is −14 and sum is −5: that is −7 and 2.
  2. Factor: (x − 7)(x + 2) = 0.
  3. Set each factor to zero: x = 7 or x = −2.

Answer: x = 7 or x = −2.

Worked Example · 03 · Absolute-value inequality

Find all x such that |x − 4| < 3.

  1. Translate to distance: "x is within 3 of 4".
  2. Write as a band: 4 − 3 < x < 4 + 3.
  3. Simplify: 1 < x < 7.

Answer: 1 < x < 7.

Explanations · with structural reasoning

Explanation · 01↑ back to Q01
Answer · A · x > −2

Subtract 5: −2x < 4. Divide both sides by −2 — divisor is negative, so flip the sign: x > −2. The flip is the only trick in this question.

Explanation · 02↑ back to Q02
Answer · D · 3

For ax² + bx + c = 0, sum of roots = −b/a = −(−3)/1 = 3. No factoring needed.

Verification by factoring: (x − 5)(x + 2) = 0 → roots 5, −2. Sum = 3. ✓

Explanation · 03↑ back to Q03
Answer · C · 1 ≤ x ≤ 4

Translate the absolute-value inequality to a band: −3 ≤ 2x − 5 ≤ 3. Add 5: 2 ≤ 2x ≤ 8. Divide by 2: 1 ≤ x ≤ 4.

Explanation · 04↑ back to Q04
Answer · A · x < −5 or x > 2

Sign chart. Roots are x = 2 and x = −5. Test regions:

x = −6:  (−8)(−1) = +8  → positive ✓
x = 0:   (−2)(+5) = −10 → negative ✗
x = 3:   (+1)(+8) = +8  → positive ✓

Take regions where the product is positive: x < −5 or x > 2.

Explanation · 05↑ back to Q05
Answer · C · |x| = |y|

x² = y² means x and y are the same distance from zero — possibly with different signs. Test x = 3, y = −3: x² = y² = 9 ✓; A fails (3 ≠ −3); D fails (3 + −3 = 0 here, but try x = 2, y = 2: x+y = 4 ≠ 0). The structurally correct statement is |x| = |y|.

Explanation · 06↑ back to Q06
Answer · C · 5

From x − y = 1: x = y + 1. Substitute into 2x + 3y = 12: 2(y+1) + 3y = 12 → 5y + 2 = 12 → y = 2. Then x = 3, so x + y = 5.

Faster path: add the two equations after multiplying. 2x + 3y = 12; multiply x − y = 1 by 2: 2x − 2y = 2. Subtract: 5y = 10 → y = 2. Same answer.

Explanation · 07↑ back to Q07
Answer · E · −8 or 8

"Exactly one real solution" means the discriminant is zero: b² − 4ac = 0. Here a = 1, b = k, c = 16, so k² − 64 = 0 → k² = 64 → k = ±8. Both values give one repeated root.

Explanation · 08↑ back to Q08
Answer · B · x² < x

Substitute x = 0.5. Then x² = 0.25, √x ≈ 0.71, 1/x = 2, x³ = 0.125.

Check each: A says x² > x — false (0.25 < 0.5). B says x² < x — true (0.25 < 0.5). C says 1/x < 1 — false (2 > 1). D says √x < x — false (0.71 > 0.5). E says x³ > x — false.

Structural insight: between 0 and 1, raising to a power shrinks the number, and dividing by it grows. This is the most-tested non-intuition in GMAT algebra.

Explanation · 09↑ back to Q09
Answer · A · −6

Take square roots, remembering the ±: x + 3 = ±5. So x = 2 or x = −8. Sum = 2 + (−8) = −6.

Vieta shortcut: expand to x² + 6x + 9 = 25, i.e. x² + 6x − 16 = 0. Sum of roots = −b/a = −6. Same answer, no listing required.

Explanation · 10↑ back to Q10
Answer · C · 4ab

Expand both squares: (a+b)² = a² + 2ab + b²; (a−b)² = a² − 2ab + b². Subtract: (a² + 2ab + b²) − (a² − 2ab + b²) = 4ab.

Pattern to memorize: (p + q)² − (p − q)² = 4pq. Recognizing this expression on sight saves 30 seconds.
